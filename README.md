# portal_through_web
This is a cinematic, fully interactive single-page web experience that transforms the history of the internet into a personal time machine. Instead of simply viewing the evolution of the web, users step into it — exploring how the internet looked, felt, and behaved across different eras.
The experience begins in a minimal void, where users are prompted to select a moment in time using immersive, tactile interfaces — a rotary year dial and a dynamic calendar. Once a date is chosen, a dramatic portal animation transitions them into a fully reconstructed version of the web from that era.

Each historical phase (1991–2025) is recreated as a living environment, with authentic design systems — from early text-based pages and chaotic GIF-heavy layouts to Web 2.0 gloss, flat design minimalism, and modern glassmorphism. Every detail evolves dynamically: typography, layout, interaction patterns, cursor behavior, and visual identity.

A personalization layer enhances the experience by relating the selected year to the user’s birth year, turning the journey into a reflective narrative rather than just a visual exploration.

The final “Mirror” act presents a recursive browser view of the experience itself, delivering a subtle philosophical message: every interaction on the web is part of a larger human story.

Built entirely using pure HTML, CSS, and vanilla JavaScript, the project showcases advanced frontend techniques such as:

Custom physics-based UI interactions (rotary dial with inertia)
CSS 3D transformations (calendar page flips)
Clip-path animations (portal tearing effect)
Dynamic theme switching across eras
Real-time cursor-based element repulsion 
Canvas-based particle systems

First Click is not just a website — it is an emotional, nostalgic, and introspective journey through the evolution of the digital world, designed to make users feel the passage of time through interaction.
